TESTS
-----
Default (no Hugging Face model download; stable on minimal CI):

  cd c:\Users\siddu\OneDrive\Desktop\llm_emotions_tribev2
  python -m pytest -q

Verbose:

  python -m pytest -v --tb=short

Uncertainty metrics + WebSocket plumbing (needs PyTorch importable):

  python -m pytest tests/test_uncertainty_signals.py tests/test_ws_plumbing.py -q

TRIBEv2 surrogate encoder tests (numpy-only):

  python -m pytest tests/test_tribe_encoder.py -q

Full integration (downloads distilgpt2; requires working PyTorch / HF):

  CMD:
    set RUN_HF_TESTS=1
    python -m pytest -q

  PowerShell:
    $env:RUN_HF_TESTS="1"; python -m pytest -q


REST SMOKE (requires API running)
---------------------------------
PowerShell example (omit model to use DEFAULT_CAUSAL_LM — tiny HF regression checkpoint):

  Invoke-RestMethod -Method POST -Uri http://127.0.0.1:8010/generate `
    -ContentType "application/json" `
    -Body '{"prompt":"Hello","max_new_tokens":8,"detect_suppression":true}'

Optional coherent English (needs RAM / HF download):

    -ContentType "application/json" `
    -Body '{"model":"distilgpt2","prompt":"Hello","max_new_tokens":8,"detect_suppression":true}'


DOCKER COMPOSE
--------------
From project root:

  docker compose build
  docker compose up

API:    http://localhost:8000
Dashboard (nginx): http://localhost:8080

Rebuild dashboard with custom WebSocket URL for other hosts:

  $env:VITE_WS_BASE="ws://YOUR_HOST_IP:8000"; docker compose build dashboard

Dashboard demo (Playwright) - optional
--------------------------------------
Requires: pip install playwright; python -m playwright install chromium

With API + Vite already running, drive one streamed generation and save a screenshot to docs\dash_demo_last_run.png:

  PowerShell (example: API on 8010, Vite chose 5174 because 5173 was busy):

    $env:ANIMA_API_PORT="8010"
    $env:ANIMA_DASH_PORT="5174"
    $env:ANIMA_DEMO_HEADLESS_ONLY="1"
    python scripts\run_dash_demo.py

  Env vars (ANIMA_* preferred; LLM_AFFECT_* still accepted):
    ANIMA_DEMO_MODEL - default hf-internal-testing/tiny-random-gpt2 (tiny RAM; distilgpt2 if you have memory).
    ANIMA_DEMO_HEADLESS_ONLY=1 - skip headed Chromium.

  Match dashboard\.env VITE_API_HTTP_TARGET to uvicorn (default http://127.0.0.1:8010). In dev, WebSockets use the Vite proxy unless VITE_WS_DIRECT=1 (then set VITE_WS_BASE).


Notes
-----
- Skipped tests: tests\test_hooks.py and tests\test_api.py unless RUN_HF_TESTS=1.
